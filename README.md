## 🛠️ 使用指南 (Friendly Guide)

### 1. 环境配置
本项目依赖 Python 3.9+ 及部分第三方库。

**安装依赖:**
使用常用的requirements.txt记录依赖,你可以像下面这样启动venv并安装需要的库
```bash
python -m venv .venv
.venv\Scripts\activate
(.venv) pip install -r requirements.txt
```

### 2. 运行代码
- **普通 Python 脚本 (.py):**
  在终端中运行，例如：
  ```bash
  python lab-08/1.py
  ```
- **Jupyter Notebook (.ipynb):**
  推荐直接在 VS Code 中打开 `.ipynb` 文件即可运行。VS Code 会自动检测并提示安装必要的内核扩展。
