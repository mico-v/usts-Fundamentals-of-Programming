1、用函数实现求解一元二次方程的根。
编写一个Python程序Solve.py。定义两个函数：第一个函数为求根函数solve(a, b, c)，参数a,b,c为一元二次方程的系数，函数返回为2个值，即方程的根，如果无实根，则返回(None, None)，如果有两个相等的根，则返回2个值相等；第二个函数为判定式函数，可使用匿名函数来计算b2-4ac。主程序：用户通过键盘输入一元二次方程的系数a、b、c，用逗号分隔，调用前面定义的solve函数，根据返回值输出3种根的情况。
2、哥德巴赫猜想。
编写一个Python程序Goldbach.py，验证“任何一个大于2的偶数总能表示为两个素数之和”。例如：24=5+19，其中5和19都是素数。定义两个函数：第一个函数isPrime(n)用于判断一个整数是否为素数；第二个函数TwoPrimes(n)用于将整数n拆分成两个素数之和，返回值为两个素数。主程序：调用函数将4~100之间的整数拆分成两个素数，并逐个打印成类似“24=5+19”这样的结果。
