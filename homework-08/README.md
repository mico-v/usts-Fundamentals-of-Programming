1．	学生成绩管理系统：
处理班级学生的三门课程（数学、语文、英语）成绩，要求提供这些功能：成绩的输入、修改、查询、输出和保存到文件。各个功能分别用函数实现，在主程序中显示一个带序号的文字菜单，根据输入的数字调用对应的功能。菜单内容如下所示：
1-输入成绩
2-修改成绩
3-查询成绩
4-输出成绩
5-保存成绩
0-退出程序
- 输入成绩时逐行输入学号、姓名和三门课成绩，以逗号分隔，空行结束输入。
- 修改成绩时先输入要修改学生的学号查询，然后输入新的成绩。
- 查询成绩时输入学号，然后显示出学号、姓名和三门课成绩。
- 输出成绩时以表格方式显示，第一行显示：学号 姓名 数学 语文 英文，后面每一行显示一个学生的成绩，按学号从小到大顺序排列。
- 保存成绩时以CSV格式保存。
